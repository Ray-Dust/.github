# 拾貳光塵
> **慎始敬終**

</div>

<div align="center">

**拾貳光塵**於2025年，由 *Steverjava* 和 *anjisuan608* 帶頭創立。
一直以來，拾貳光塵秉持着尊重任何一個**作品**，並且尊重每一個**作者**的大前提進行着原創與二創。
我們始終相信，**優秀**的作品不應被埋沒，而優秀的作品在國內會有語言的障礙，這是拾貳光塵創建的初衷，也是我們不願意看到且痛心的地方。

拾貳光塵當然不止步於漢化某些內容，**雅**才是我們想要達到的目標。
而**其他項目**也正在被提上日程
